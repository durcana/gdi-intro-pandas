What is Pandas?

  • pandas is a Python package providing fast, flexible, and expressive data structures designed to make working with relational data both easy and intuitive. It is an extremely useful tool for doing practical, real world data analysis in Python.

Why Pandas?

Adding and Removing Columns

You can quickly delete columns and add new columns to your dataframe

Deleting columns is done with the command del

You can also create a new column by simply adding two existing columns together


In [8]: purchases['total_price'] = purchases['price'] + purchases['tax']

In [9]: purchases.head()

In [10]: del purchases['tax']

In [11]: purchases.head()

Notice how referencing a dataframe column looks a lot like referencing a dictionary key

Fill in empty cells

If your data came with empty cells, pandas will fill it in with NaN.

This can cause an error if you are trying to add two integer columns together.

To solve this, we can use the fillna() command to replace NaN with whatever we choose.

Notice that I refer to the first_name column as purchases.first_name. That is also accepted syntax in pandas.


In [12]: purchases.first_name.fillna(' ')

Sorting Your Data

You can sort your data by any row

Sorting will default to ascending=True, so if you want to sort descending, you have to specify in your parameters ascending=False

Let's take a quick look at the pandas documentation for more information


In [13]: purchases.sort_values(by='date', index=False)

In [14]: purchases = purchases.sort_values(by=['date', 'price'], ascending=False,
index=False)

Changing a Row's Data Type

When importing data, you may see your totals column imported as a string, but you need it to be a float.

pandas as a function astype() that lets you change the data type of your column.

Let's try turning our number of visits into a string, and then into a float


In [16]: purchase.visits = purchases.visits.astype(str)

In [17]: purchase.visits = purchases.visits.astype(float)

Column to Python List

You may come across situations where you want one column as a python list to use for outside analysis.

pandas makes it easy with a tolist() function


In [18]: ids_list = purchases.member_id.tolist()

In [19]: len(ids_list)

Dataframe Manipulation: Filtering

pandas filtering can look a little funny, but it is extremely useful

Let's look at the following code and break down exactly what is happening


In [20]: large_purchases = purchases[purchases['price'] > 100.0]

You want to look from inside out

We are looking at the price column

We are only looking at rows where the price column is greater than 100 dollars.

We are keeping all of purchases where price is greater than 100 dollars.

Concatinating DataFrames

To concatinate dataframes means to take two dataframes with the same columns and putting together the rows

Let's import the csv purchases-october.csv

Since this csv has the columns as purchases, we can concatinate the two dataframes like so:


In [21]: pandas.concat(purchases, purchases-october)

Merging DataFrames

You can think of merging dataframes a lot like SQL joins

You can use a column (or a list of columns) to merge two dataframes together.

Let's merge purchases and members on the column member_id


In [22]: merged = purchases.merge(members, how='left', on='member_id')

The how parameter is comparable to the type of join you would use in SQL.

on decides the column or columns (as a list) that you want to merge on

Group By

groupby allows you to group by 1 or more columns to get a more top-level view of your data

There are a couple of ways to do this, but here is one recommended way.

Either way, you will need the as_index=False


In [23]: prices_avg_by_date = purchases.groupby('date', as_index=False)[['price', 'total_price']].avg()

In [24]: total_by_member_and_date = purchases.groupby(['date', 'member_id'], as_index=False)['total_price'].sum()
